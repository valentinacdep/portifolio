Plataforma VOVOTINA

Auxílio aos idosos

Desenvolvimento de uma plataforma voltada a cuidados com idosos.

Vovotina é o meu TCC: uma plataforma criada para facilitar a vida dos idosos, com ferramentas simples para organizar rotinas, cuidados e lembretes. Desenvolvido com carinho, o Vovotina torna o dia a dia mais leve, seguro e acolhedor para quem precisa.
